## 1. Frequently Asked Questions

### What is "The Project Maverick"?

The Project Maverick is a real-time stock price prediction platform. It uses XGBoost machine learning models to ingest live market data and forecast where a stock's price is likely to move during the next trading day. The system provides 26 predicted price points covering the full trading day at 15-minute intervals, along with confidence scores for each prediction.

### Where does the data come from?

The backend integrates with Financial Modeling Prep (FMP), a financial data provider that supplies accurate market data including open, high, low, close, and volume (OHLCV) data matching real-world stock exchanges.

### How many stocks are tracked?

The system currently tracks 29 stock symbols including major companies like AAPL, MSFT, GOOGL, AMZN, TSLA, NVDA, META, JPM, and others.

### Why does the prediction line sometimes stay flat?

This means the stock market is currently closed (weekends, evenings, or holidays). The system pauses inference when no new trading data is available. Predictions will resume when the market reopens.

### What do the technical indicator abbreviations mean?

| Abbreviation | Full Name | What It Measures |
|-------------|-----------|-----------------|
| RSI | Relative Strength Index | Momentum on a 0-100 scale. Above 70 = overbought, below 30 = oversold |
| MACD | Moving Average Convergence Divergence | Trend-following momentum from two moving averages |
| ATR | Average True Range | Price volatility over a given period |
| OBV | On-Balance Volume | Cumulative buying/selling pressure based on volume |

### How accurate are the predictions?

Prediction accuracy varies by stock and market conditions. The system provides a confidence score with each prediction to help you gauge reliability. Historical prediction performance can be compared against actual outcomes through the dashboard.

## 2. Platform Navigation

### How to select a stock

1. Look at the sidebar on the left side of the dashboard.
2. Click on any stock symbol (e.g., AAPL, TSLA).
3. The main chart area will update to show that stock's data and predictions.

### How to read the chart

- **Solid lines/candlesticks:** Historical actual price data from real market trading.
- **Dashed or highlighted overlay:** Predicted future price path generated by the ML model.
- **Hover tooltips:** Place your cursor on any data point to see the exact price, date, and time.

### How to access the API documentation

Navigate to `http://localhost:8000/docs` (or your deployed backend URL with `/docs`) to view the interactive Swagger UI. This shows all available API endpoints with request/response examples.

## 3. Troubleshooting

### The page loads but shows no chart data

The frontend and backend are separate services. If the page layout renders but shows loading indicators:

1. Verify the backend is running (check `http://localhost:8000/docs`).
2. Confirm the `VITE_API_URL` environment variable in the frontend points to the correct backend address.
3. Check that the PostgreSQL database is running and contains market data.

### Missing data for certain dates

If certain dates show gaps in the chart, this typically means the market was closed on those dates (weekends, holidays) or the data polling service had not yet fetched that data. The system will fill gaps automatically during the next polling cycle.

## 4. Understanding Confidence Scores

Each prediction point includes a confidence score:

| Range | Interpretation |
|-------|---------------|
| Above 0.80 | High confidence - the model found strong patterns matching historical training data |
| 0.60 to 0.79 | Moderate confidence - reasonable prediction with some market uncertainty |
| Below 0.60 | Low confidence - current market conditions differ significantly from training data; treat with caution |
